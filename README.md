## How to Use This Workspace

1. **Explore a project:**
   ```bash
   cd <project-folder>
   npm install
   npm run dev  # or relevant command
   ```

2. **Add a new experiment:**
   - Create a new folder in `applied_engineering/`
   - Add its own `package.json`
   - Document purpose in this README

3. **Reference structure:**
   - Check existing projects for patterns
   - Reuse configurations as templates
